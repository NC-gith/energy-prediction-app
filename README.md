# ⚡ Energy Prediction App

A cloud-deployed web application that predicts total electricity consumption using a Google Vertex AI AutoML Regression model.

---

## Overview

This project demonstrates the end-to-end deployment of a machine learning application on Google Cloud Platform.

The application provides an interactive Streamlit interface where users can enter electricity-related information and obtain an estimated total electricity consumption (kWh).

---

## Features

- Interactive Streamlit web application
- Electricity consumption prediction
- Google Vertex AI AutoML Regression
- Docker containerization
- Google Cloud Run deployment
- Continuous deployment with Cloud Build
- GitHub integration
- IAM authentication
- Responsive user interface
